My favourite HTML tags

Here are my top 5 HTML tags in no particular order!

html
The tags you place at the begining and end of an HTML file.
head
Provides information such as the title.
body
Contains the contents of the document.
hr
The Thematic Break, creates a horizontal line.
ol
The ordered list tag. The tag is used to order a numbered list of items.
